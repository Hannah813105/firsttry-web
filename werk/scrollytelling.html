Halloween scrollytelling

Een geanimeerde website over de geschiedenis van Halloween
Apr-Jun 2023

Tijdens het vak van Front-end development heb ik gezamenlijk met Quinty Kuiper en Tirza Meiema een character index voor de game deltarune gemaakt.

Hiervoor hebben we het voor het eerst het framework van React JS gebruikt en zelf een JSON file met arrays van objecten in elkaar gezet met gebruik van een excel sheet.

Mijn bijdrage binnen dit project is deel van het excel sheet, verbeteringen in de JSON file en het design het het toevoegen van een css grid aan de html.
